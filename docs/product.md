# Product Direction

Internal target: Siri-class local personal assistant.

Working product name: opensiri-ai.

Public wording: opensiri-ai, an open-source Mac personal assistant harness.

## Principles

- Local-first.
- Harness-primary safety.
- Per-source opt-in context.
- User approval before destructive or external actions.
- Inspectable audit trail.
- Memory should explain where answers came from.

## UX

- Spotlight-style command palette.
- Conversation view.
- Source chips for memory results.
- Approval sheet for risky actions.
- Replayable transcript.
- Permission center.
- Finder Quick Action target via `scripts/install-finder-quick-action.sh`.
